## Color Usage Rules

- **Contrast**: All text must meet WCAG AA minimum (4.5:1 for body, 3:1 for large text)
- **Meaning**: Use color consistently to encode meaning — green=success, red=error, amber=warning
- **Mix on components**: Combine primary + neutral backgrounds, use accent sparingly for CTAs and highlights
- **Readability first**: Never sacrifice readability for aesthetics; test light text on colored backgrounds
