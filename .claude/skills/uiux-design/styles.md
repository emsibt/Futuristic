# Style Library

## 1. Clean Modern
- Generous whitespace, flat design, subtle shadows
- Card-based layouts with rounded corners (8-12px)
- Minimal borders, rely on elevation and spacing for separation
- **Best for**: Learning platforms, productivity tools, SaaS dashboards

## 2. Soft Glassmorphism
- Frosted glass effect with `backdrop-filter: blur()`
- Semi-transparent backgrounds with subtle borders
- Layered depth with overlapping translucent panels
- **Best for**: Creative apps, portfolios, modern landing pages

## 3. Vibrant Playful
- Bold rounded shapes, large border-radius (16-24px)
- Prominent use of primary colors, illustrated accents
- Bouncy animations, oversized interactive elements
- **Best for**: Educational apps for younger audiences, gamified platforms

## 4. Professional Minimal
- Sharp lines, restrained color use, structured grid
- Typography-driven hierarchy, monochrome with one accent color
- Subtle transitions, no decorative elements
- **Best for**: Enterprise tools, admin panels, data-heavy interfaces

## 5. Warm Friendly
- Soft rounded corners, warm tones, approachable feel
- Gentle gradients, inviting illustrations or icons
- Comfortable spacing, easy on the eyes for extended use
- **Best for**: Learning platforms, community apps, wellness/health tools
